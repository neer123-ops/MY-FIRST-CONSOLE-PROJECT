My first project on console

Steps to set up Visual Studio Code

1. First download the Visual Studio code App
2. Create a new folder and name it what ever you want
3. Then open the Visual Studio Code App and click on files, got to 'Open Folder... Crtl + K + Crtl + O' and open the folder which you created.
4. Now right click on the blank black space and choose 'Create new file'
5. Now Create three new files- 'index.html', 'main.js', 'style.css' and start your coding!
6. For previewing the app you have created, first go to files and choose 'Save all', and then go to the html screen and right click, and choose 'open with live server.'

How to open console screen

1. Create your app on the Visual Studio Code app and open it with live server
2. Now right click on the screen and choose 'inspect'
3. Then on the top you will see a button named 'CONSOLE'
4. Click on that button and you will see the console screen
5. The console screen will tell you what values have been stored
6. It will also tell what mistakes you have made in your code; like typing errors or undentified things e.t.c
